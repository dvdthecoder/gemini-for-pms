# Module 3 — Vibe Coding

**3 lessons · ~2 hours**

Module 3 teaches you to build working PM tools without writing code. By the end you'll have built two tools and know how to publish them as shareable URLs.

---

## What is Vibe Coding?

Vibe coding is building software by describing what you want in plain English. You are the PM — you define the outcome, review what was built, give feedback, and decide when it's done. Gemini writes the code.

You don't need to understand the code. You need to know if it works.

---

## What You'll Build

**Lesson 3.1** — The Stride Activation Dashboard: a working HTML page that reads product metrics and surfaces insights, ready to share with your CEO.

**Lesson 3.2** — Plan your own tool: choose a real PM problem, write a brief, prepare the build spec.

**Lesson 3.3** — Build and publish it: ship your planned tool and make it available as a URL.
