# Module 1 — Fundamentals

**7 lessons · ~3 hours**

Module 1 teaches the core skills you'll use in every Gemini CLI session. By the end you'll be comfortable loading files, prompting well, researching in real time, and carrying your context between sessions.

## How to Start

Open Gemini CLI from the course folder and type:

```
/start-1-1
```

The lessons run in sequence. Complete each one before moving to the next — later lessons build on earlier ones.

!!! tip "Already done Module 0?"
    Type `/course-map` to see your progress and jump directly to your next lesson.
